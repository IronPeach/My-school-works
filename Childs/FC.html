HTML&MD ~ The root of a website!


    HTML, or HyperText Markup Language, file extension name is .html, makes up most of the website that exist in the internet. Try to do Shift + Ctrl + I, or Ctrl+Shift+J, you can see that this website is full of html. The HTML is the outline of all the programming language of website (JS, CSS, Node.JS, JQuery, PHP, etc...), because you plug those code in the HTML. The HTML is made of tags and tags, a really easy structure! A normal HTML will look like this:
<h1>This is a simple text.</h1>
    We can see that there is things like <h1></h1>, that is a tag. A tag is “a marker used to create HTML elements. All elements begin with a start tag, and most end with an end tag. These tags determine what type of element is created, and the start tag include attributes for the element”(Morgan, p.303) For example, <h1> is a start tag, </h1> is a end tag. Think about it like this: All the code of HTML is like a hamburger, you want to make it the way you want, so you need to use different ingredient, the code between is like the meat and lettuce and the ingredient you want... You design it! And the Tag is like the bun, it goes in front and the end of the code, just like the bun of burger goes top and bottom(Please donate me a burger! I'm so hungry!). You can add Attribute in the start tag:
<h1 style = "color:#66ccff">This Text will have #66ccff as color</h1>
<h2 style = "color:red">This Text will have red as color</h2>
    Now, we can see that They got attributes! The style attribute can control the style of a text, like align, color, highlight, ect... And once you start learn HTML, there is a gonna be a tons of attribute and tag that you need to learn, and the syntax that html require, it’s kind of hard if you are weak at logical, but it’s fine, we got a new way to write HTML!
    MD, or MarkDown, file extension name is .md, is used for make writing HTML easier. The creator of MarkDown, which is John Gruber, want it be “as readable as possible”[1], so this MarkDown things comes out, it is very readable, clear, you can plug in HTML in there too. Which is awesome! This is what a normal MD file looks like:
##This context will be identical with the one below.##
<h2>This content will be identical with the one above</h2>
    We can see, this thing is just a easy way to write html! This is a how a MarkDown work. Now we know what a simple HTML look like, but those are casual way to write, if you’re gonna write your code like that when you’re working, your boss is gonna fire you. You need to have a formal way to write your code. There is a structure for a HTML document:
