PHP ~ Easy Learning - Hard Mastering


    PHP, or the PHP: Hyper Preprocessor (I don't know what’s the point of PHP in front) is a back-end language that is usually used for creating database, things like register, login, recording data. It’s one of the most popular server side language that is really easy to learn, but really hard to remember, so that's why many people put their PHP book aside while they code PHP(=_=). A normal PHP can be plug-in into the html, or it can be a file, its extension name will be .php or .php3, .php4 and more. A normal piece or the Hello world of PHP will look like this:
<?=echo "Hello world"
    Yes, very ugly. In the code, echo is the same thing as display, it means that display a Hello world text on website, the reason why Hello world is in "" is because its a string, a string means a bunch of letter and numbers and character, it’s usually quoted(except html). But you can also just embedded your code in the html, like this:
<!DOCTYPE html>
<html>
<body>
         echo "This is another string!"
    //And now, this is a comment that can include whole line.
    /* This comment style can last many lines.
    *Like this
    *And this
    */You can make this as long as you want.
    ?>
 </body>
</html>
    We can see that the php code is embedded between the <?php & ?>, this is how it's embedded. The php is very powerful and easy to learn, and it get the same salary as the other language, the time like if you need money, you could learn php, and go to work, it could work. But be careful, PHP is easy to learn, but hard to master, because the basic thing can do whatever you needed, so many PHP user doesn’t have the skill that they supposed to have. This is how PHP look like, easy to learn, powerful, you could've use it wisely, but if you don’t like this, it's fine, we still got the others!
